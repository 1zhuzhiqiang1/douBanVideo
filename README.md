##Angular项目实战

运行方式
```angular2html
git clone https://github.com/zhuzhiqiang/douBanVideo.git
cd douBanVideo
npm i
npm run start

```

大家也可以直接从 release 下载 douBanVideo.zip 压缩包，直接解压缩到 tomcat 的 webapps 目录下，
访问地址 http://localhost:8080/douBanVideo，其中端口地址改成你自己的
